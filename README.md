# CLI-TUBE
A simple youtube viewer, I hope you like it.

## Dependencies
```
curl
mpv
yt-dlp
```

## Installation

Clone the repo and go to the folder:
```
cd cli-tube
sudo make install
```

If you want to uninstall it just execute
```
sudo make uninstall
```

## Usage
```
$ cli-tube <query>
```
